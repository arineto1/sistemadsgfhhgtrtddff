Gerenciamento de Pastas e Arquivos

Tutor Especialista: Foco em Professor de Geografia - ETEC/CEETEPS

1. Definição e Estrutura de Hierarquia

No Windows, a organização é hierárquica e em árvore. Tudo começa na Unidade Raiz (ex: C:\).

  • Arquivos: Unidades lógicas de dados (documentos, imagens, executáveis).
  • Pastas (Diretórios): Containers usados para organizar arquivos e outras subpastas.
  • Caminho (Path): A "folha de rosto" da localização. Ex: C:\Projetos\Mapas\Brasil.shp.
Peguinha de Prova: O Windows não é Case-Sensitive para nomes de arquivos. Ou seja, "MAPA.jpg" e "mapa.jpg" são lidos como o mesmo arquivo na mesma pasta.

2. Regras de Nomenclatura (Mnemônico SABIDA)

O Windows impede o uso de 9 caracteres específicos. Memorize o mnemônico para não errar questões de "qual nome é válido":

\ / | : * ? " < >

  • \ / | (Barras/Pipe)
  • : (Dois pontos)
  • * (Asterisco)
  • ? (Interrogação)
  • " (Aspas duplas)
  • < > (Setas/Maior-Menor)

3. Extensões: Foco Técnico Geografia

O Windows usa a extensão para associar o arquivo a um programa padrão. No contexto de Geografia na ETEC, atenção especial a:

Extensão Tipo de Arquivo / Uso Importância para Geografia
.SHP / .DBF Shapefile (Vetorial) Obrigatório em SIG (QGIS/ArcGIS).
.KML / .KMZ Google Earth Intercâmbio de dados geográficos leves.
.TIF / .TIFF Imagem (Raster) Imagens de Satélite e ortofotos.
.EXE / .MSI Executáveis Instaladores de softwares.
.PDF Documento Portátil Relatórios técnicos finais.

4. Manipulação: O Segredo de Arrastar

Este é o tema que mais gera "peguinhas" em provas da FAT/CEETEPS:

Mesma Unidade (ex: de C: para C:): O padrão é MOVER.

Unidades Diferentes (ex: de C: para Pendrive E:): O padrão é COPIAR.

Modificadores Forçados:

  • Segurar CTRL ao arrastar: Sempre COPIA.
  • Segurar SHIFT ao arrastar: Sempre MOVE.
  • Segurar ALT ao arrastar: Cria ATALHO.

5. Sistema de Arquivos e Lixeira

O NTFS é o sistema padrão atual. O FAT32 é limitado (arquivos de no máx 4GB).

CUIDADO COM A LIXEIRA (TOP PROVA):

  • Arquivos de PENDRIVES e Cartões SD não vão para a lixeira (são excluídos direto).
  • Arquivos de Unidades de Rede também são excluídos direto.
  • Atalho SHIFT + DELETE pula a lixeira em qualquer lugar.

6. Busca Inteligente (Wildcards)

Para localizar arquivos quando não sabemos o nome completo:

  • Asterisco *: Substitui qualquer quantidade de caracteres.
    Ex: *.shp busca todos os arquivos shapefiles.
  • Interrogação ?: Substitui apenas um único caractere.
    Ex: Mapa?.jpg encontraria "Mapa1.jpg", mas não "Mapa10.jpg".

7. Atributos e Bibliotecas

Arquivos possuem "etiquetas" que definem seu comportamento:

  • Somente Leitura: Impede salvar alterações no arquivo.
  • Oculto: Não aparece na busca comum do usuário leigo.
  • Bibliotecas: Recurso que agrupa pastas em uma única visão (ex: Imagens), mesmo que elas estejam em discos diferentes. Excluir uma biblioteca não apaga os arquivos, apenas o agrupamento.

Checklist de Atalhos para Memorizar

Win + E : Abre o Explorador de Arquivos.

F2 : Renomeia o item selecionado.

Alt + Enter : Abre Propriedades (ver tamanho e atributos).

Ctrl + Shift + N : Cria nova pasta.

Win + D : Minimiza tudo e mostra a Área de Trabalho.

Ctrl + F : Inicia pesquisa na pasta aberta.
