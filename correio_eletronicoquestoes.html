Questão 01: No âmbito da infraestrutura tecnológica do Centro Paula Souza, o docente necessita compartilhar relatórios e dados geoestatísticos. Analise as proposições sobre o funcionamento técnico do e-mail:

I. O protocolo SMTP atua como a espinha dorsal para o envio de mensagens.
II. Para o recebimento, pode-se optar pelo protocolo POP3 (descarrega) ou IMAP (sincronização).
III. O protocolo HTML é utilizado obrigatoriamente como mecanismo de transporte para documentos .pdf ou .docx.

Resolução: O SMTP é para envio (Correto). O POP3 e IMAP são para recebimento (Correto). HTML é uma linguagem de formatação, não um protocolo de transporte de arquivos; documentos são transportados via extensões MIME do SMTP.

Resolução: O campo "Cco" (Blind Carbon Copy) oculta os endereços dos destinatários ali inseridos dos demais participantes. É a ferramenta correta para preservar a privacidade institucional.

Resolução: "Vossa Senhoria" é o padrão para autoridades e servidores. Termos como "Digníssimo" e "Ilustríssimo" foram abolidos da redação oficial paulista por serem considerados redundantes.
Questão 06: Sobre as recomendações para o envio de documentos anexos em correios eletrônicos oficiais, assinale a alternativa correta:
Resolução: Nunca se envia anexo sem explicação no corpo do texto. Isso garante clareza e segurança (evita que o e-mail pareça spam ou vírus).
